This guide doesn't tell you everything you need to know when it comes to Android Theming. This guide is mainly for icon pack creations unss someone is willing to help add onto it.

So you came here wanting to learn how to create your own icon pack huh and become someone big in the Play Store? Well you’ve come to the right place because I will teach you how you can make your own icon packs. I’ll teach you the things you’ll need and how to do it all step-by-step. But in order to make it anywhere in the Play Store you need to be good at graphic design you need to know how to take your time on every single icon. You also need to be patient and have the time that gets dedicated into this job, yes I said job because you can make this a job no you don’t get paid as much as you would if you had a actual job. If you try your pretty much wasting time so focus on those who are actually buying your icon packs.

 

So lets dive on into creating your 1st icon pack & the things needed in order to do this.

 

What you need list:

  • Android Studio
  • Java
  • A photo editor like Photoshop , GIMP or Paint Shop Pro
  • CandyBar or Blueprint, unless you know how to code then you can make your own dashboard
  • Some coding knoweledge

 

Some useful site you’ll need when creating icon packs:

  • Android Asset Studio
  • Batch Image Resizing Made Easy
  • Android Activities
  • Importing an SVG or PSD file

Now after you have installed the install-able items from the what you need list then you can open the photo editor of you choice and create a 512x512 new document.

Now you can make you icon base any shape or color you want it just make sure it'll look good at the end so you earn money instead of lose money.

As to how to name the icons when saving you can name them whatever you want but remember what you name the icon will show up within the dashboard of your choosing. So don't name your icons like ic_google leave the ic out below is how to name your icons if you want them to actually show what the icon belongs to in the app.

 

  1. betternet.png will become Betternet
  2. google_chrome.png will become Google Chrome
  3. pay__pal.png will become PayPal
  4. _kaip.png will become kaip
  5. ___kaip.png will become KAIP
  6. ___npr_one.png will become NPR One
  7. material___os.png will become MaterialOS

 

After you have all your icons made you want to use the Batch Image Resizing Made Easy website listed above to resize your icons to 192x192 if you size them bigger your users will have issues changing icons manually which you want to avoid at all cost.

 

Now that’s it for this tutorial if you need some pointers or any advice let me know in a reply below and good luck. I do want to add 1 more thing when you start getting icon requests always ask for a receipt number or some proof of purchase it’s to help cut back on piraters.
